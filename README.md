# 🏜️ Desert Journey Portfolio (Source)

This is the source code for my cinematic, immersive portfolio website. It is built with **React**, **Vite**, and **GSAP**.

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 2. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 3. Development
Run the development server:
```bash
npm run dev
```

---

## 🏗️ Build & Deployment

To update the live website at `mohamedali1937.github.io`:

1.  **Build the project**:
    ```bash
    npm run build
    ```
2.  **Copy the output**:
    Copy all files from the `dist/` directory to your `MohamedAli1937.github.io` repository.
3.  **Push to GitHub**:
    Commit and push the changes from the `github.io` repository.

---

## 🎨 Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Animations**: GSAP (GreenSock)
- **Styling**: Vanilla CSS (Modern design)
- **Icons**: React Icons (Feather)
